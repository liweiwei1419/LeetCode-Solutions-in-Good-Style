460. LFU缓存

LFU 是 Least Frequently Used 近期最少使用算法，在缓存满的时候，移除那个频率最小的 k-v 对。

LRU 算法是首先淘汰最长时间未被使用的页面，而 LFU 是先淘汰一定时间内被访问次数最少的页面。
